## Architecture

Never:

- Duplicate logic
- Break layered architecture
- Put product code into Core

Always:

- Follow ADRs
- Follow Roadmap
- Follow Sprint Scope

## Quality

No feature is complete until:

- Tests pass
- Review completed
- Documentation updated
- Repository clean
